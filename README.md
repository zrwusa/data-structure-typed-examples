# data-structure-typed-examples

## Installation

For development

```sh
// with npm
npm install
npm run dev

// with yarn
yarn
yarn dev
```

For simulating a web server serving path '/public' as a web at local.

```sh
// with npm
npm run build
npm run dev:serve

// with yarn
yarn build
yarn dev:serve
```
